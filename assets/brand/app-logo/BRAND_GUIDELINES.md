## Brand Overview

### Brand Name
**Venus One** (Product) | **AstroBoBo** (Company)

### Brand Promise
*Your cosmic guide to astrology and self-discovery*

### Brand Personality
- **Mystical** — Ancient wisdom meets modern insight
- **Warm** — Inviting, nurturing, supportive

## Logo

### Primary Logo: Venus Symbol

The Venus logo features a stylized Venus symbol (♀) rendered in a pearl/blush gradient, representing femininity, beauty, and cosmic connection.

#### Logo Construction

```
┌─────────────────────────┐
│                         │
│       ╭───────╮         │
│      ╱ Pearl   ╲        │  ← Circle: Pearl/Blush gradient
│     │  Gradient │       │     (#FFF5F0 → #D8968E)
│      ╲         ╱        │
│       ╰───┬───╯         │
│           │             │  ← Stem: Solid blush
│         ──┼──           │  ← Cross: #C88A82
│           │             │
│                         │
└─────────────────────────┘
```

#### Master Colors

| Element | Start | End | Use |
|---------|-------|-----|-----|

## Imagery & Icons

### Photography Style

- **Celestial**: Night skies, stars, galaxies, cosmic phenomena
- **Natural**: Crystals, flowers, natural elements with mystical quality
- **Lifestyle**: Aspirational moments of reflection and self-discovery
- **Color treatment**: Warm shadows, purple/gold highlights

### Illustration Style

- **Geometric**: Clean zodiac symbols, constellation patterns
- **Ethereal**: Soft gradients, particle effects, light rays
- **Mystical**: Ancient symbols modernized with gradients

### Iconography

- **Style**: Outlined, 2px stroke weight
- **Corner radius**: 2px on internal corners
- **Color**: Inherit from context (gold for features, white for navigation)

## Voice & Tone

### Brand Voice

| Attribute | Description | Example |
|-----------|-------------|---------|
| Mystical | Tap into cosmic wisdom | "The stars have aligned..." |
| Warm | Supportive, nurturing | "Your journey continues..." |
| Empowering | Build confidence | "Embrace your fire sign energy" |
| Clear | Accessible astrology | "Mercury retrograde means..." |

### Writing Guidelines

- **Be inclusive**: Use "you" and "your" — never assume gender
- **Be positive**: Focus on growth opportunities, not doom
- **Be specific**: Reference actual planetary positions
- **Be concise**: Mobile-first content, scannable

### Zodiac Descriptions

Each sign should be described with:
1. Element connection (Fire/Earth/Air/Water)
2. Ruling planet influence
3. Core personality traits (positive framing)
4. Current cosmic energy

## Usage Guidelines

### Logo Don'ts

| Don't | Why |
|-------|-----|
| Stretch or distort | Maintains brand integrity |
| Change colors arbitrarily | Pearl/blush is core identity |
| Add effects (shadows, outlines) | Logo is self-contained |
| Place on busy backgrounds | Ensures visibility |
| Rotate the logo | Venus symbol has fixed orientation |

### Color Don'ts

| Don't | Why |
|-------|-----|
| Use low-contrast combinations | Accessibility violation |
| Mix warm/cool tones inappropriately | Breaks visual harmony |
| Override planet colors | Users expect consistency |

### Typography Don'ts

| Don't | Why |
|-------|-----|
| Use fonts outside the stack | Brand consistency |
| Set body text below 14px | Readability |
| Use all-caps for body copy | Readability |
